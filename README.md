<p align="center">
  <strong>RBX-Fileview</strong>
</p>

Human-readable, diff-friendly dumper for Roblox place and model files.

## Installation

### [GitHub Releases](https://github.com/Barocena/RBX-Fileview/releases/latest)

Pre-built binaries are available on the [releases page](https://github.com/Barocena/RBX-Fileview/releases).

### [Rokit](https://github.com/rojo-rbx/rokit)

```bash
rokit add Barocena/RBX-Fileview
```

## Usage

```bash
rbx-fileview dump place.rbxl
```

## VS Code Extension

Install the extension from vsix file on the [releases page](https://github.com/Barocena/RBX-Fileview/releases) (coming to vscode marketplace soon)
